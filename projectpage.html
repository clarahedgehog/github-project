Remote repositories are where your code is stored, and are necessary in collaborative coding. When you work with other people, you use remote repositories to push and pull data that you need when sharing work.

How to create a project on GitHub

Show the steps for creating a project on GitHub.

The Git commands to create a project locally

    Create a local directory:

  • mkdir my-project
  • cd my-project

    • Initialize a Git repository to turn the project in to a Git project:

  • git init

    • Create files:

  • echo/touch "my-project" README.md

    • Add files:

  • git add . or git add "filename" for specific file

    • Commit files:

  • git commit -m "initial commit"

Link a local repo to a remote repo

    Create a repository on GitHub:

  • Go to your GitHub account and select "New repository". Give it a name and choose public.

    • Link the local code to the remote repo:

  • git branch -M main
  • git remote add origin "https://..."
  • git push -u origin main
